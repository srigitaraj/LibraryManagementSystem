This screen represents the Main Menu of the Library Management System, built to manage book records efficiently.

Features Shown

Add a Book

Allows the user to add a new book to the system by entering:

ISBN Number

Book Name

Book Type

Author Name

Book Cost

On clicking AddBook, the book details are stored and processed by the backend servlet.

Search Book

Provides an option to search for a book using its ISBN code.

On successful search, the system displays:

Book Title

Author Name

Author Contact

Book Price

Book ISBN

Technical Flow

The application runs on Apache Tomcat (localhost:8080).

Requests are handled by a MainServlet.

Data is dynamically retrieved and displayed based on user input.

This interface acts as the core interaction point for adding and retrieving library book details in a simple and user-friendly manner.
